# 前言

在看python代码时经常看到这样的代码

```python
print(r"aa\nbb")
print(u"aa\nbb")
```

字符串使用时直接引用就好了，前面的r和u这两个前缀是什么鬼？有什么作用？

纸上得来终觉浅，绝知此事要躬行。

要了解r和u的作用还是实际操作来看看。

# 说明

- r的作用

  首先看看r的作用，执行如下代码看看结果：

  ```python
  print("aa\nbb")
  print(r"aa\nbb")
  ```

  打印结果是这样子：

  ```
  aa
  bb
  aa\nbb
  ```

  第一个打印，**\n**把字符串截开了，而第二个打印是全部打印。

  在python中，**\n**是换行符作用。所以第一个打印才是正常的。

  第二个打印中的**r作用就是取消字符串中的转移字符的作用，让其老老实实作为一个普通字符串**。

- u的作用
